# QA - Casos de Teste Manuais

Este repositório contém **casos de teste manuais** criados para validar funcionalidades comuns em sistemas web, com foco na **qualidade do software**, regras de negócio e experiência do usuário.

O material aqui representa minha prática e evolução contínua na área de **Quality Assurance (QA)**.

---

## Objetivo
- Demonstrar conhecimento em **testes manuais**
- Praticar a criação de **casos de teste bem estruturados**
- Validar fluxos funcionais de sistemas
- Aplicar boas práticas de QA utilizadas no mercado

 ## Funcionalidades testadas
-  Login
-  Cadastro de usuários
-  Pagamento com cartão
-  Pagamento via PIX
-  Outras funcionalidades serão adicionadas gradualmente

##  Organização do repositório
Os casos de teste estão organizados por funcionalidade, utilizando arquivos em **Markdown**, simulando planilhas de testes usadas no dia a dia de QA.

Cada caso de teste contém:
- ID do caso de teste  
- Cenário  
- Pré-condição  
- Passos para execução  
- Resultado esperado  
- Prioridade
